## 📌 What It Does

A full-stack MERN application where developers paste any code snippet and receive an AI-generated review covering bugs, performance issues, code style, and improvement suggestions — powered by Google Gemini.

---

## ✨ Key Features

- **Multi-language support** — Works with JavaScript, Python, TypeScript, and more
- **Instant AI analysis** — Gemini API reviews code for bugs, inefficiencies, and best practices
- **Clean editor UI** — Syntax-highlighted code input with clear results display
- **Review history** — Past reviews saved and accessible via MongoDB

---

## 🛠️ Tech Stack

React · Node.js · Express.js · MongoDB · Google Gemini API

---

## 🚀 Getting Started

```bash
git clone https://github.com/Ali-Raza-12/code-reviewer.git
cd code-reviewer

# Frontend
npm install
npm run dev

# Backend (in /server)
cd server
npm install
npm start
```

Add your `GEMINI_API_KEY` and `MONGODB_URI` to `.env`.
